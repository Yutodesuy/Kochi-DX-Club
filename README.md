# Kochi-DX-Club

高知高専で立ち上げる「高知DX部」の静的トップページです。

## Files

- `index.html`: ホームページ本体
- `styles.css`: レイアウトとデザイン
- `script.js`: メニュー開閉とスクロール演出
- `favicon.svg`: アイコン

## Preview

ブラウザで `index.html` を直接開くか、簡易サーバーで確認できます。

```powershell
py -m http.server 8000
```

## Note

応募導線の Googleフォーム URL は `script.js` の `GOOGLE_FORM_URL` に設定する想定です。
